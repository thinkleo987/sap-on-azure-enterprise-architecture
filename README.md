# SAP on Microsoft Azure Enterprise Architecture Handbook

## Purpose

This repository defines enterprise architecture patterns for SAP workloads on Microsoft Azure.

It aligns SAP design with:
- Azure Well-Architected Framework
- Azure Landing Zone
- Cloud Adoption Framework
- SAP Notes and SAP standards
- Microsoft Azure architecture guidance

## Scope

Covers enterprise SAP workloads:
- SAP ECC
- SAP S/4HANA
- SAP BW/4HANA
- SAP BusinessObjects
- SAP HANA databases
- SAP NetWeaver systems
- SAP on Linux and Windows

## Architecture Domains

- Landing Zone design
- Networking architecture
- Identity and access
- Compute architecture
- Storage architecture
- High availability
- Disaster recovery
- Security architecture
- Operations
- Monitoring
- Cost governance
- Performance engineering

## Design Principles

- Design for resiliency
- Use Azure Landing Zones
- Automate everything
- Enforce security by default
- Use SAP certified infrastructure only
- Separate environments
- Monitor continuously
- Optimize cost continuously
